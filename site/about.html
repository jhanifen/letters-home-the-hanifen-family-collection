How These Letters Found Their Way Online

Sometime in the early 2020s, Bob Hanifen Jr. handed over a box of letters to his cousin Jim. They had been written decades earlier by his father, Bob Sr., along with Bob's brothers Jim and John and their father James. The letters spanned from the late 1930s through the mid-1940s — years that carried the boys from home through college and into the Second World War.

The letters had survived in remarkably good condition, but they were handwritten in the flowing cursive of another era, and there were a lot of them. Transcribing 84 letters by hand — some running more than a dozen pages — was a project that could easily take months. So Jim tried a different approach.

Built with AI

This entire project was built with the help of Claude Code, Anthropic's AI coding assistant, and the Claude API. The pipeline works like this:

1
Scan

Each letter was carefully scanned, producing 304 page images from 84 letters.

2
Transcribe

Claude's vision model read each handwritten page and produced structured data — the full text, plus metadata like sender, recipient, date, and location.

3
Analyze

The full collection was analyzed to build a narrative timeline, infer missing dates, and identify the story arc that connects all 84 letters.

4
Generate

From the structured data, the code generated this interactive website, a typeset PDF memoir with embedded scans, and a narrated audiobook.

Every script in the pipeline — the transcription, the analysis, the web and book generators — was written collaboratively with Claude Code. The AI didn't just process the letters; it helped build the tools to process them.

Open Source

The full source code for this project is available on GitHub. It includes the Python pipeline scripts, the structured letter data, and the web output. The original scans are too large for a code repository but the transcriptions and metadata are all there — enough for someone to pick up where this left off, or adapt the approach for their own family's letters.

If your family has a box of old letters gathering dust somewhere, maybe this can show what's possible.
